# Smart Path
# Περιγραφή
Η κατασκευή smart path είναι ένα έξυπνο μπαστούνι, σχεδιασμένο για να επιτρέπει την εύκολη και ασφαλή πλοήγηση των τυφλών ατόμων ή των ατόμων με περιορισμένη όραση μέσα στην πόλη.
Η κατασκευή μας ερείδεται στην εικόνα που έχουμε από την πόλη μας, όπου στα πεζοδρόμια και στους πεζόδρομους βρίσκονται αντικείμενα (π.χ. δέντρα), η ύπαρξη των οποίων πολλές φορές συνιστά μείζονα δοκιμασία για έναν άνθρωπο που δεν μπορεί να τα εντοπίσει εγκαίρως.
Το smart path είναι ένα “έξυπνο μπαστούνι” για την επίλυση του παραπάνω προβλήματος. Για να επιτευχθεί αυτός ο στόχος, στην άκρη του μπαστουνιού που είναι πιο κοντά στο έδαφος είναι τοποθετημένος ένας αισθητήρας απόστασης. Χάρη στην παρουσία αυτού του αισθητήρα και με την βοήθεια ενός vibration motor ο χρήστης του μπαστουνιού ειδοποιείται για όποιο τυχόν εμπόδιο στην διαδρομή του, έχοντας ως αποτέλεσμα την αποφυγή ατυχημάτων.
Θέλοντας να προεκτείνουμε την λύση που δίνουμε στο πρόβλημα, προσθέσαμε στο μπαστούνι ένα αισθητήρα GPS και ένα επιταχυνσιόμετρο, ώστε να εντοπίζονται οι δύσβατες διαδρομές, και ένα Bluetooth Module, με το οποίο οι μετρήσεις των δύο παραπάνω αισθητήρων στέλνονται σε ένα app για την αξιολόγηση τους βάσει αντικειμενικών κριτηρίων και την δημιουργία ενός χάρτη με διαδρομές ευχέρως προσβάσιμες σε άτομα με οπτική αναπηρία. 
Παρόλα αυτά, στον δρόμο του χρήστη μπορεί να βρεθούν προσωρινά αντικείμενα, όπως για παράδειγμα ένας περαστικός ή κάποιο σκυλί, τα οποία καταχωρούνται στην εφαρμογή λανθασμένα ως μόνιμα εμπόδια και υπάρχει ο κίνδυνος να θεωρηθεί έτσι μια πλήρως κατάλληλη διαδρομή μη κατάλληλη. Για την αποφυγή αυτού του είδους των σφαλμάτων, έχουμε βάλει στην λαβή του μπαστουνιού ένα switch button, με το οποίο το άτομο μπορεί, από την στιγμή που πέσει στην αντίληψη του, να δηλώσει ένα εμπόδιο προσωρινό.
